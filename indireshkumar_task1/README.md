# Voice Assistant

## Objective
Build a Python-based voice assistant that listens to spoken commands and performs basic voice-controlled tasks such as greeting the user, telling the current date and time, and performing web searches.

## Features
- Capture voice input using a microphone
- Respond to greetings with text-to-speech
- Tell the current date and time
- Perform web searches using the default browser
- Handle unrecognized speech gracefully
- Provide voice feedback for all responses using text-to-speech

## Technologies Used
- Python
- SpeechRecognition
- PyAudio
- pyttsx3
- webbrowser
- datetime

## Skills Demonstrated
- Speech recognition
- Text-to-speech
- Voice command processing
- Exception handling
- Browser automation
