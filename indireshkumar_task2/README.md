# BMI Calculator

## Objective
Develop a command-line Python application that calculates a user's Body Mass Index (BMI) and classifies it into standard health categories based on the user's height and weight.

## Features
- Accept user weight and height as input
- Calculate BMI using the standard formula
- Display BMI rounded to two decimal places
- Categorize BMI as:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Validate user input
- Handle invalid and negative values gracefully

## Technologies Used
- Python

## Skills Demonstrated
- User input handling
- Conditional statements
- Functions
- Arithmetic operations
- Exception handling
