# Random Password Generator

## Objective
Create a Python application that generates secure random passwords based on user-defined requirements such as length and character types.

## Features
- User-defined password length
- Minimum password length validation
- Choose character types:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Ensure at least two character types are selected
- Generate strong random passwords
- Generate multiple passwords without restarting the program

## Technologies Used
- Python
- random
- string

## Skills Demonstrated
- Randomization
- Input validation
- Loops
- Conditional logic
- Function design
