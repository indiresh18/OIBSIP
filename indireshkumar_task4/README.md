# Basic Weather App

## Objective
Build a Python application that retrieves and displays real-time weather information for a specified city using the OpenWeatherMap API.

## Features
- Search weather by city name
- Retrieve live weather data using an API
- Display:
  - Temperature (°C and °F)
  - Humidity
  - Weather description
  - Wind speed
- Handle invalid city names
- Handle invalid API keys
- Handle network errors
- Validate empty user input

## Technologies Used
- Python
- Requests
- OpenWeatherMap API

## Skills Demonstrated
- REST API integration
- JSON parsing
- HTTP requests
- Exception handling
- Input validation
