# Chat Application

## Objective
Develop a real-time command-line chat application that allows two users to exchange messages over a local network using Python socket programming.

## Features
- Server accepts incoming client connections
- Client connects to the server
- Real-time bidirectional communication
- Timestamped chat messages
- Graceful client disconnection handling
- Runs locally using localhost
- Supports two connected users

## Technologies Used
- Python
- socket
- threading
- datetime

## Skills Demonstrated
- Socket programming
- Client-server architecture
- Multithreading
- Network communication
- Exception handling
